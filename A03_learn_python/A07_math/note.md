## 基本概念
标量：只有大小没有方向的量，我们生活中常见的1、2、3

向量：具有大小和方向的量，在数学中，通常表示一个有序数字列表，如二维、三维空间中的点

矩阵：矩阵是一个二维数组，其中的每个元素都是一个标量。它可以表示线性变换。

张量：一个可以在多个方向有分量的数学对象。标量是0阶张量，向量是1阶张量，矩阵是2阶张量。更高阶的张量可以看作是多维数组
